Supplementary metadata

Creative Work Info

This provides some supplementary metadata properties used to describe documents, in particular PDFs in XMP packets.

URI: http://www.creativeworkinfo.org/ns/core

Recommended namespace: cwic

Titles and Names

surtitle

Lang Alt
Defines the surtitle (the title excluding the subtitle, if any) of the document. This is the prefix of or equal to the value of dc:title in Dublin Core.

subtitle

Lang Alt
Defines the subtitle of the document, if any. This is usually the suffix of dc:title, but some creators put a version number at the end of the dc:title value.

franchise

Lang Alt
The name of the franchise for this document, e.g. "Tolkien's Middle Earth"

series

Lang Alt
The name of the series for the document, e.g. "The Lord of the Rings"

Credits and Attribution

shortcreator

Text
Abbrieviated name of the creator(s), e.g. "Carrol, L"

excludeURI

Text
URI pointing to a description of the content which is excluded from the overall license of the document in some fashion.

contributor

Seq of Text
Those who intellectually contributed to this work, e.g. "John Tenniel, Illustrator; Jessie Wilcox Smith, Illustrator"

supporter

Seq of Text
Those who offered non-intellectual support for this work, e.g. "Alice Pleasance Liddell, Inspiration; Lenny de Rooy, Layout"

contactURI

Text
The URI (mailto: or http:) used to get in touch with those responsible for the document, e.g. the rights holder(s).

publisherURI

Text
The URI of the publishing entity, if any, e.g. to its web site.

publisherLocality

Text
Publisher's location (geographically), e.g. "London", "Westminster, London"

Referencing and Ordering

isbn

Text
The ISBN number of the document, if any.

sourceDOI

Text
The DOI (Digital Object Identifier) for the source of this work (if any), as provided by DOI.org registrion agencies such as Zenodo.org.

datasetDOI

Text
The DOI (Digital Object Identifier) for the dataset repository associated with this work (if any).

datasetRepositoryURI

Text
The URI to the dataset repository associated with this work (if any), such as a .zip file or a source code repository.

volume

Text
The volume number of the document.

issue

Text
The issue number of the document.

version

Text
The version number of the document.

originalDocumentID

Text
The document ID of the original document, if any. This property must be preserved at all times, and must be transferred over into consecutive exports of the same document. When a document is derived, this value should be copied into parentDocumentID, and the document ID of the derivative work should be placed here instead.

parentDocumentID

Text
The document ID of the document from which the original document was derived, if any.
